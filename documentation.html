Preparation

Before you proceed with HTML Multipurpose Template installation, you need to prepare. It is recommended to follow steps, listed below:

Software

We recommend that you download all the required software to edit website template. The software required is listed on the template preview page. Technical requirements might differ depending on the exact template, therefore we will list the most basic ones:

  1. You should also use code editor, i. e. Sublime Text, Notepad++, Visual Studio code, etc. for template files source code editing.
  2. For files uploading to the server FTP manager will be also required, like FileManager at your hosting control panel or desktop applications, such as Total Commander, FileZilla, CuteFTP, etc.
Hosting

You can just run an index.html file directly from your site folder, but it won't let you explore all template functionality. You should upload a template to the hosting server in order to display a fully working website correctly.

You may also install the Website Template to your computer locally by using localhost. In order to set up a local server, you may use WAMP, AppServ, MAMP, or any similar software. All of them are installed the same as a regular application.

Feel free to check the following tutorials on how to set up local server:

  • How to install AppServ web development environment
  • How to install WAMP web development environment
  • How to install XAMP web development environment
Template Upload

You need to upload all files of site/ folder from your localhost to hosting.

Note!

Note: the root directory of a site depends on the internal structure of your hosting server. If you upload the site folder itself on hosting, the root of your site will be http://your_domain_name/site. You need to upload inner content of site/ folder if you want url to be http://your_domain_name/
Warning: do not upload sources / folder, or any of the source template files to your hosting server.

Template Structure

We follow the standard method and format for the developed HTML pages. We put comments on each section's purpose in HTML, CSS & JS files. We try to optimize coding as soon as possible.
